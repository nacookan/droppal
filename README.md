# DropPal

DropPal は、二者間でブラウザだけでサーバーを経由せずにファイルやメッセージを直接やりとりできるツールです。

アカウント登録やインストールは不要です。相手と同じ時間に画面を開き、招待URLと返信コードをやりとりすると接続できます。

## 使い方

1. 招待する人: DropPal を開いて「はじめる」を押す
2. 招待する人: 表示された招待URLをコピーして相手に送る
3. 招待される人: 届いた招待URLを開く
4. 招待される人: 表示された返信コードをコピーして相手に送る
5. 招待される人: 招待ページに戻って待つ
6. 招待する人: 受け取った返信コードを貼り付けて「接続する」を押す
7. 接続されたら、メッセージやファイルを送受信できる

## 注意事項

- 二人が同時にオンラインである必要があります
- 通信環境によっては接続できない場合があります
- 画面を閉じたり接続が切れたりした場合は、最初からやり直してください
- 大きすぎるファイルはうまく送れない場合があります

## 仕組み

DropPal は WebRTC でブラウザ同士を直接接続します。

接続に必要な情報は Vanilla ICE でまとめ、LINE やメールなど別の連絡手段でコピー&ペーストしてやりとりします。これにより、専用のシグナリングサーバーを用意せずに使えます。

通信経路の確認には、以下の STUN サーバーを使用しています。

- `stun:stun.cloudflare.com:3478`
- `stun:stun.l.google.com:19302`

## デプロイ

静的ファイルを公開できる場所に配置すれば動作します。

GitHub Pages、Cloudflare Pages、Netlify などの静的ホスティングに置くほか、レンタルサーバーに FTP でアップロードするだけでも公開できます。

## ローカルで動かす

```sh
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```
